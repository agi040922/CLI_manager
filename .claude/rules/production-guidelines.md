# Production Product Guidelines

이 프로젝트는 프로덕션 제품이다. 다른 사람도 사용하므로 아래 사항을 반드시 고려한다.

## 개인정보 보호

- 개인정보(이름, 경로, API 키 등)를 코드에 하드코딩하지 않는다
- 사용자별로 다를 수 있는 값은 설정(Settings)으로 분리한다

## 다양한 환경 고려

- 내 환경을 기준으로 하되, 다른 사람 설정이 다를 수 있음을 고려한다
- 설정이 다를 경우에도 정상 작동해야 한다(앱이 멈추면 안된다.)
- 필요시 Settings에서 설정할 수 있는 옵션을 검토한다

## 코드 변경 시 영향 분석

- 기능 추가/수정 시 해당 부분만 보지 않는다
- 함수 재사용, 다른 코드에서 참조하는 부분을 꼼꼼히 확인한다
- 변경이 다른 기능에 영향을 주는지 반드시 검토한다(다른 기능에 영향이 간다면 검토를 하고 허락을 맡는다)

## 작업 완료 후 보고

- 작업 후 다양한 유저 시나리오를 검토한다
- 오류 가능성이 있는 부분을 보고한다
- 변경으로 인한 사이드 이펙트를 최소화한다

## 릴리즈 전 체크

- 릴리즈 전에 모든 변경사항이 커밋되었는지 확인한다
- 커밋이 원격 저장소에 푸시되었는지 확인한다
